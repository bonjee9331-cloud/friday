# FRIDAY

Ben Lynch's personal AI brain. Built by merging BOB v5 (D2MS sales ops assistant) and Job Autopilot into one Next.js app. Runs on Netlify with Supabase for memory. One brain. Many bodies.

## What it does

- **Chat with voice** ... type or talk to Friday in the browser. Built in Web Speech API for input, browser TTS or ElevenLabs for output. Works on desktop and mobile.
- **BOB sales ops** ... D2MS HelloFresh floor dashboard with rep stats, daily briefings generated by Claude, coaching focus.
- **Job Autopilot** ... the full original job-autopilot module is preserved (jobs, applications, interviews, brain analysis, packages, settings).
- **Daily tasks** ... assign Friday any task, it will run it via the brain and report back.
- **Telegram bot** ... same brain, accessible from your phone via Telegram webhook. Use `/bob`, `/jobs`, `/task` prefixes to switch modules.
- **Persistent memory** ... every conversation is saved to Supabase and loaded back in so Friday remembers.

## Quick deploy (Netlify + Supabase)

### 1. Create a Supabase project

Go to supabase.com, create a project, copy the project URL and the anon + service role keys.

In the SQL editor, paste the contents of `docs/schema.sql` and run it. This sets up every table Friday needs.

### 2. Deploy to Netlify

```bash
# From this folder
netlify deploy --prod
```

Or connect the repo in the Netlify UI. The `netlify.toml` already points at Next.js.

### 3. Set environment variables

In Netlify > Site settings > Environment variables, add everything in `docs/ENV_VARS.md`. The bare minimum is:

- `ANTHROPIC_API_KEY`
- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE_KEY`

### 4. Open your site

You'll land on the chat page. Talk to Friday.

### 5. Optional Telegram hookup

Create a bot with BotFather, grab the token, then set your webhook to:

```
https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://<your-netlify-site>/api/friday/telegram?token=<TELEGRAM_WEBHOOK_SECRET>
```

Ping it from your phone. Friday replies.

## What was merged from where

**From BOB v5 (`bob-ai-main.zip`):**
- The BOB v5 master system prompt (now in `lib/friday-prompt.js`)
- The D2MS rep data and patterns (now in `components/BobDashboard.js`)
- The ElevenLabs TTS voice (now in `/api/friday/voice`)
- The conversation memory concept (now persisted in Supabase)

The original Python `bob.py` is not used... it was a local Whisper + sounddevice loop which will not run on Netlify. The Web Speech API replaces Whisper for browser voice. If you want the local desktop voice loop as well, keep running `bob.py` locally in parallel and point it at the `/api/friday/chat` endpoint... you will get the same brain with the local mic.

**From Job Autopilot (`job-autopilot-main.zip`):**
- The full Next.js 14 app structure
- The Supabase client setup
- The original `runBrain` dual LLM pattern (extended in `lib/brain.js`)
- All jobs, applications, interviews, brain, packages, settings pages preserved as is
- The job sources, fit scoring, cover letter generation API routes

## Security note

The original `bob.py` had API keys hardcoded in plain text. Rotate them now. None of the new code holds any secrets... everything reads from env vars.
